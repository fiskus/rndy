Rndy

The right way to manage passwords!

Forget about keeping your passwords on hard drive - it's not secure! Generate them from one master-passwords and three differend salt's.

Get code on GitHub

Easy to install

There are packages for Ubuntu/Debian and Archlinux.
See details on Installation section of wiki.

Easy to access

You can use as CLI version on you PC, as web-version in your browser. If you haven't access neither program, neither Internet you can still reproduce actions of script manually.

Secure to use

With this tool you will have one strong password per account. Passwords will not store on your hard drive, so no one can access to them.

Usage

$ rndy -u username -d example.com -o
Type master-password: …typing…password…
MTY1NDRkMTY1NDRkRkMT

 

$ rndy -u username -d example.com -p Jfsi43 -c 8 -o
MTY1NDRk

Tool named in honor of Randall "Randy" Lawrence Waterhouse from Cryptonomicon.
